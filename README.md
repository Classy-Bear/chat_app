# ChatApp

A simple chat app. This project is under development.

## Getting Started

Install dependencies.
```bash
flutter pub get
```
Run project
$1 = Url of the REST API
$2 = Url of the Socket API
```bash
flutter run --dart-define=API_URL=$1 --dart-define=SOCKET_URL=$2
```
